### 🗺️ PLANGO – 사용자 맞춤 여행 일정 추천 서비스
> 복잡한 여행 계획을 자동화하는 **개인화 여행 일정 생성 플랫폼**

- 🧭 **여행 일정 자동 생성 알고리즘**
  - 사용자 여행 목적지·기간·관심 카테고리 기반 일정 자동 생성
  - 국내 여행 정보를 제공하는 *TourAPI*, *여기어때*, *롯데관광 API* 데이터 연동
  - 카테고리(자연·체험·식사 등) 분류 모델로 목적지 필터링 및 우선순위 반영

- 🧠 **LLM 기반 POI 설명 요약 & 최적 동선 구성**
  - 목적지 리뷰·설명 데이터를 기반으로 LLM이 중요 정보를 요약하여 사용자 이해도 향상
  - CoT(Chain of Thought) 기반 추론으로 “설명 → 특징 요약 → 일정 반영” 프로세스 자동화  
  - LangGraph 기반 워크플로우로 여행 일정 자동 작성 파이프라인 구성

- 🗂️ **시각적 일정 관리 UI**
  - 여행 일정 요약 페이지, 추천 스팟 페이지, 상세 일정 지도 기반 페이지 제공
  - 지도에서 동선 확인 가능, 일정 요소별 편집·추가·삭제 제공

- 🛰️ **데이터 파이프라인 & 백엔드 구조**
  - 여행지 데이터 수집 → LLM 기반 요약 → 일정 생성 → 시각화까지 하나의 파이프라인으로 구성
  - RAG 기반 검색으로 더 정확한 여행지 정보 반영

- 🔗 Repo: [`plango`](https://github.com/plango2025)
